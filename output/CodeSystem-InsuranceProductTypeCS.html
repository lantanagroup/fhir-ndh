CodeSystem: Insurance Product Type Code System

Official URL: http://hl7.org/fhir/us/ndh/CodeSystem/InsuranceProductTypeCS Version: 1.0.0
Active as of 2022-10-31 Computable Name: InsuranceProductTypeCS

A distinct package of health insurance coverage benefits that are offered using a particular product network type.

This Code system is referenced in the content logical definition of the following value sets:

  • InsuranceProductTypeVS

This code system http://hl7.org/fhir/us/ndh/CodeSystem/InsuranceProductTypeCS defines the following codes:
