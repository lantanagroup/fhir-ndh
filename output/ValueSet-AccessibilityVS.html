ValueSet: Accessibility Value Set

Official URL: http://hl7.org/fhir/us/ndh/ValueSet/AccessibilityVS Version: 1.0.0
Active as of 2022-10-31 Computable Name: AccessibilityVS

Codes for documenting general categories of accommodations available

References

  • Accessibility

Logical Definition (CLD)

  • Include all codes defined in http://hl7.org/fhir/us/ndh/CodeSystem/AccessibilityCS

 

Expansion

This value set contains 7 concepts

Expansion based on Accessibility Code System v1.0.0 (CodeSystem)

CodeSystemDisplayDefinition
  cultcomphttp://hl7.org/fhir/us/ndh/CodeSystem/AccessibilityCSCultural competence

Provides services that meet the social, cultural, and linguistic needs of persons.

  handiaccesshttp://hl7.org/fhir/us/ndh/CodeSystem/AccessibilityCShandicap accessible

  cognitivehttp://hl7.org/fhir/us/ndh/CodeSystem/AccessibilityCScognitive

Explanation of the columns that may appear on this page:

Level A few code lists that FHIR defines are hierarchical - each code is assigned a level. In this scheme, some codes are under other codes, and imply that the code they are under also applies
System The source of the definition of the code (when the value set draws in codes defined elsewhere)
Code The code (used as the code in the resource instance)
Display The display (used in the display element of a Coding). If there is no display, implementers should not simply display the code, but map the concept into their application
Definition An explanation of the meaning of the concept
Comments Additional notes about how to use the code
