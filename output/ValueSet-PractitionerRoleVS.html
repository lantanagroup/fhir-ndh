ValueSet: PractitionerRole Code Value Set

Official URL: http://hl7.org/fhir/us/ndh/ValueSet/PractitionerRoleVS Version: 1.0.0
Active as of 2022-11-04 Computable Name: PractitionerRoleVS

Codes for the capabilities that an individual, group, or organization is acknowledged to have in a payer network, including general codes from the HL7 PractitionerRole Code System

References

  • NDH Base PractitionerRole
  • NDH Exchange PractitionerRole

Logical Definition (CLD)

This value set includes codes based on the following rules:

  • Include all codes defined in http://hl7.org/fhir/us/ndh/CodeSystem/ProviderRoleCS
  • Include all codes defined in http://terminology.hl7.org/CodeSystem/practitioner-role
