Logical Definition (CLD)

This value set includes codes based on the following rules:

  • Include all codes defined in http://hl7.org/fhir/organization-role
  • Include all codes defined in http://hl7.org/fhir/us/ndh/CodeSystem/OrganizationAffiliationRoleCS

This value set excludes codes based on the following rules:

  • Exclude these codes as defined in http://hl7.org/fhir/organization-role

    memberMemberA type of non-ownership relationship between entities (encompasses partnerships, collaboration, joint ventures, etc.)
